Overview

Object representing a “LineItem”

Examples:

LineItem.new({
  :name => "Ham and Eggs by dr. Seuss",
  :quantity => 1,
  :unit_of_measure => "book",
  :description => "The famous childrens book",
  :image => "http://blogs.slj.com/afuse8production/files/2012/06/GreenEggsHam1.jpg",
  :price_per_unit => 1299
})
